## Antes de atualizar!

Existem alguns arquivos que é bom salvar antes de atualizar:

### Onde achar esses arquivos?

No Prism Laucher, você pode clicar com o botão direito sobre uma instância e escolher "Folder", isso vai abrir a pasta da instância no seu explorador de arquivos.

Aberto a pasta, vá em .minecraft, é aqui onde está os arquivos

### Quais são os arquivos?

<Card title=".sl_password" icon="document">
    Esse arquivo armazena sua senha de verificação com o server, visto que o server suporta jogadores de offline, essa senha previne usar o nick de outro jogador, perder essa senha é problematico visto que você não conseguira entrar no server!
</Card>

<Card title="options.txt" icon="document">
    Esse arquivo armazena suas preferências de jogos pessoais, como controles, volume, e gráficos.
</Card>

<Card title="/local" icon="document">
    É uma pasta, ela armazena seu mapa e waypoints! Seu mapa sera zerado sem essa pasta
</Card>


### Outros arquivos interessantes (opcionais)

<Card title="/saves" icon="document">
    Essa pasta armazena seus mundos, se tiver algum que queira salvar.
</Card>

<Card title="/schematics" icon="document">
    Essa pasta armazena suas esquemáticas do Create, caso tenha alguma.
</Card>


### O que fazer com esses arquivos?

Copie ou recorte para outro lugar, por exemplo, sua área de trabalho, eles ficarão aqui temporariamente.


## Atualizando

Após salvar os arquivos importantes, basta seguir [Esses passos](/guias/0-install/#importando-o-zip) para importar o .zip novo, **depois volte aqui.**

Isso vai ter criado uma instância nova com a versão atualizada.

### Restaurando os aquivos salvo

Após criar a nova instância, use os passos anteriores para chegar até a .minecraft da instância nova. 

Pegue os arquivos que você salvou e cole aqui. Se tiver algum aviso de arquivos de mesmo nome, substitua todos.

## Conclusão

Inicie o jogo, se tudo der certo, não tera a tela para criar senha e no jogo você tera seu mapa.

**Divirta-se**
